## 🚀 The Vision
Cities grow fast, and keeping track of infrastructure isn't easy. **CivicEye** bridges the gap between citizens and the City Council. If you see a pothole, a broken streetlight, or a water leak, you pin it on the map. The city sees it, prioritizes it, and fixes it. Simple.

## ✨ The "Wow" Features

* **🗺️ Interactive Precision Mapping:** Powered by Leaflet.js, citizens can drop precise incident pins anywhere in the city.
* **🔥 Dynamic Priority Heatmaps:** Issues aren't just listed; they glow. The map generates live heatmaps based on citizen upvotes to show the city exactly where the "hot spots" are.
* **👍 The "Me Too!" Engine:** A built-in upvote system lets the community back up existing reports, ensuring critical issues get fixed first.
* **🌍 Bilingual UX:** Built-in instant translation toggles the entire platform between **English** and **Kinyarwanda** for true local accessibility.
* **🏛️ Mayor's Operations Center:** A secure admin backend where city officials can review photographic evidence, track emails, and dispatch "Resolved" status updates.

## 🛠️ The Tech Stack

**Frontend Architecture:**
- Pure HTML5 & CSS3 (Custom Glassmorphism UI)
- Vanilla JavaScript (Asynchronous Fetch API)
- Leaflet.js (Interactive Mapping Engine) & Leaflet.heat Plugin

**Backend & Data:**
- PHP (RESTful API Design)
- MySQL (PDO structured, relational database)
